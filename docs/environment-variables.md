# Environment variables

| Variable       | Description       | Default value | Expected value                      | Required |
|----------------|-------------------|---------------|-------------------------------------|----------|
| `DEBUG`        | Enable debug mode | `false`       | `true` or `false`                   | no       |
| `LOGGER_LEVEL` | Logger level      | `info`        | `debug`,  `info`, `warn` or `error` | no       |
